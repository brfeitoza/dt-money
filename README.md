# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/brfeitoza/dt-money.git && cd dt-money

# Install Dependencies
$ yarn

# Run Aplication
$ yarn start
```
Go to http://localhost:3000/ to see the application running.
